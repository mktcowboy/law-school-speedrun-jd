# Subject Progress Checklist Template

One per subject. Mark a subject `done` only when every box is real, not aspirational. Pair this with the [study system](../00-start-here/04-study-system-and-deliverables.md) and the [landmark case canon](../00-start-here/10-landmark-case-canon.md).

## Subject

## Reading

- [ ] read one full open casebook or substantial free text
- [ ] read the controlling rules, statute, or constitutional provisions
- [ ] read the landmark cases for this subject in the case canon

## Synthesis

- [ ] subject outline (`5 to 15` pages), organized by doctrine
- [ ] attack outline (`1 to 2` pages) for timed use
- [ ] rule sheet (one-line rules only)
- [ ] can explain the major doctrinal buckets out loud without notes

## Writing And Assessment

- [ ] wrote at least one timed `IRAC` or issue-spotter for this subject
- [ ] compared one answer against a model and logged the gaps
- [ ] error log started and reviewed

## Canon Check

- [ ] can state the rule each landmark case stands for in one sentence
- [ ] flagged the remaining gaps for a slower second pass

## Status

- not started / in progress / first pass solid / exam-ready
