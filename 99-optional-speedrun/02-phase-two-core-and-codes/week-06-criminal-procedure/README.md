# Week 6: Criminal Procedure

## Focus

Study the constitutional limits on policing and prosecution.

## Outcomes

By the end of the week, you should be able to explain:

- the Fourth Amendment search-and-seizure framework
- warrants and exceptions
- exclusionary rule basics
- Fifth Amendment interrogation principles
- Miranda
- Sixth Amendment right to counsel basics

## Primary Reading

- H2O Criminal Procedure: <https://opencasebook.org/casebooks/8116-criminal-procedure/>
- Congress.gov Constitution Annotated: <https://constitution.congress.gov/>
- Oyez: <https://www.oyez.org/>
- CourtListener for full opinions: <https://www.courtlistener.com/>

Cases to know:

- `Mapp v. Ohio`
- `Terry v. Ohio`
- `Katz v. United States`
- `Miranda v. Arizona`
- `Gideon v. Wainwright`

## What To Build

- one `Fourth Amendment` trigger chart
- one `warrant exception` sheet
- one `Miranda` checklist
- one `right to counsel` timeline

## Suggested Daily Split

- Day 1: what counts as a search or seizure
- Day 2: warrants and probable cause
- Day 3: exceptions and exclusionary rule
- Day 4: custodial interrogation and Miranda
- Day 5: charging, counsel, and prosecution-stage rights
- Day 6: synthesis writing

## Writing Assignment

- Write one search-and-seizure issue spotter.
- Write one interrogation/right-to-counsel issue spotter.

## If You Buy One Supplement

- use open materials first
- buy a criminal procedure supplement only if Fourth Amendment doctrine is not clicking

## Week Exit Standard

You can explain how the Constitution regulates state power before trial.
