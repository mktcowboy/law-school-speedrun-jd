# Optional Speedrun Layout

This folder preserves the original compressed structure for people who want a strict timeline.

It is now secondary, not primary.

## Who Should Use This

- people who already understand the purpose of the core subjects
- people building a crash-course map before deeper study
- people who want a time-boxed boot camp after finishing the main year-based curriculum

## What It Does Well

- forces momentum
- groups related doctrines efficiently
- gives a hard-edged study cadence

## What It Does Poorly

- it can make people think finishing a week means mastering a subject
- it is less faithful to the ordinary `1L/2L/3L` sequencing of law school
- it encourages compression before comprehension if used too early

## Recommended Use

Use the traditional curriculum first:

- [01-1l-foundation-year](../01-1l-foundation-year/README.md)
- [02-2l-doctrinal-depth-and-codes](../02-2l-doctrinal-depth-and-codes/README.md)
- [03-3l-practice-synthesis-and-launch](../03-3l-practice-synthesis-and-launch/README.md)
- [open resource library](../00-start-here/resources/open-resource-library.md)
- [source priority rule](../00-start-here/resources/source-priority-rule.md)

Then use this speedrun folder if you want:

- a second pass
- a review boot camp
- a compression challenge

For materials, let the main `1L` and `2L` reading guides control. This folder is mainly for sequence and pacing, not for overriding the newer source-priority rules.

## Layout

- `01-phase-one-foundations`
- `02-phase-two-core-and-codes`
- `03-phase-three-advanced-and-litigation`
