# Evidence

## Why It Belongs In 2L

Evidence makes more sense once you already know how cases are built and argued. It is doctrine plus courtroom filtering.

## Master These First

- relevance

Read especially:

- `FRE 401`
- `FRE 403`
- `FRE 404`

- `FRE 803`
