# Start Here

This folder is the launch dashboard for the curriculum. Use it to choose a route, learn the basic legal-method skills, and find resources only when you need them.

## Fastest Path

1. Read the [quick start](./quick-start.md).
2. If case reading is new, do the [methods bridge](./methods/README.md).
3. Pick a subject from the [subject index](../SUBJECTS.md).
4. Use the [study system](./resources/study-system-and-deliverables.md) to turn reading into outlines, briefs, and short answers.
5. Add timed practice with the [assessment method](./assessment/exam-archive-method.md).

## What Lives Here

- [methods](./methods/README.md) - opinion reading, briefing, synthesis, outlining, and short IRAC answers
- [resources](./resources/open-resource-library.md) - open casebooks, school-produced guides, sourcing rules, books, supplies, and supplements
- [assessment](./assessment/exam-archive-method.md) - practice questions, self-grading, and the landmark case canon
- [pathways](./pathways/study-pathways.md) - bar-core, litigation, transactional, and constitutional-law routes
- [maintenance](./maintenance/feedback-integration-plan.md) - project planning notes that are not part of the learner path
