# Top-10 School Priority Basis

Verified on `June 2, 2026`.

This curriculum now uses a `top-10-school-only` rule for school-produced resources.

That means:

- if a resource is coming from a law school, it should come from the latest publicly available `top-10` law-school band
- if a top-10 school does not offer a strong free public resource for the subject, use the `official source of law` instead
- inside that top-10 band, give first priority to `Ivy League schools`

## Current Ranking Basis

This rule is based on the latest publicly available summaries of the `2026 U.S. News` law-school rankings released on `April 7, 2026`.

Because there is a three-way tie at `No. 9`, the `rank 1-10 band` currently contains `11 schools`.

## How The Curriculum Uses This

When choosing among free public materials:

1. prefer `Yale`, `Penn`, `Harvard`, and `Columbia` if they offer a strong public resource
2. if the Ivies do not offer a good public option for that subject, use another school inside the same current `top-10` band
3. if the real target is the law itself, move to the `official text` instead of forcing a school source
